# Solvys UI Detail

Improve perceived quality without adding ornament. Load `/solvys-feels` and `reference/design-guidelines.md` first.

## Checklist

- Hierarchy: primary action/readout obvious under three seconds.
- Alignment: edges, baselines, icons, text, and numeric columns are optically stable.
- Typography: tabular numbers, balanced headings, readable line length.
- Surfaces: purposeful glass, no nested cards, concentric radii.
- Motion: interruptible, contextual, no decorative shimmer/sparkle/bounce.
- States: loading, empty, error, disabled, hover, focus, selected.
