# Recordly Shooting Plan — Fintheon "Infinite Situational Awareness"

Companion to the script at `~/.claude/plans/ask-me-questions-iterative-panda.md`.

**Strategy:** record body blocks first while the desk is hot, save typography-only blocks (A, B, D, G, K, L, M) for last — those are black-frame title cards, do them in one quiet sitting. Group surface visits by **app focus** so you don't bounce between Consilium → TradingView → Consilium → TradingView. Recordly auto-zooms on click — so what you click IS the emphasis. Plan your clicks accordingly.

---

## Session order (record in this sequence)

### Pass 1 — Live Fintheon surfaces (one Consilium session, don't quit)

**1. Block I — Arbitrum + TradingView (90s only · 14s)**
Do this **first** because it's the most fragile (needs a fresh verdict + TradingView logged in + chart synced).

- Pre-roll: fire `POST /api/arbitrum/deliberate`, wait for verdict, open TradingView Desktop dark-theme on the same ticker, snap windows side-by-side.
- Start Recordly. Frame the split.
- **Click 1 (zoom anchor):** the **consensus probability digit** in Doto on the Arbitrum panel. Recordly will punch in on the digit as it ticks — that's the hero shot.
- **Click 2:** the dissent line as it writes itself. Auto-zoom holds context.
- **Click 3:** swing cursor across to the TradingView candle that lines up with the verdict's IV sim. Auto-zoom pulls right, sells "the chart, right there."
- Stop record. Name `block-I.mp4`.

**2. Block H — RiskFlow sheriff tease (60/90 · 12s)**
Now that the worker's hot, fresh wires are scrolling.

- Open Strategium drawer to RiskFlow at full height.
- Start Recordly. Don't click anything for the first 4 seconds — let the auto-scroll be the hero. Cursor parked off-frame.
- **Click 1 (mid-clip):** click the **top wire's IV badge**. Auto-zoom punches in on the score — that's the "speed flex." Hold 1s.
- **Click 2:** click the next wire below as it cascades. Two clicks in 3 seconds = velocity feel.
- Stop record. Name `block-H.mp4`.

**3. Block F — NarrativeFlow card → cluster drawer (60/90 · 10s)**
Cluster must have ≥6 events. Pre-pick it before you roll.

- Position cursor near the target card before starting the recorder so the entry isn't a mile-long mouse path.
- Start Recordly.
- **Click 1 (THE shot):** the NarrativeFlow card. Recordly auto-zooms on the card; the `t-panel-slide` drawer enters from the right. This click is the centerpiece of the whole 90s — make sure it's calm and centered.
- **Click 2:** the topmost event in the drawer to highlight a timestamp. Auto-zoom on the timestamp sells "every wire that touched it."
- **Scroll** the drawer slowly with the wheel — Recordly tracks the scroll, no click needed.
- Stop record. Name `block-F.mp4`.

**4. Block E — Sanctum DayCard + NarrativeFlow (60/90 · 8s)**
Same Consilium, just shift focus left.

- Verify today's day plan is posted and streak digit is non-zero.
- Start Recordly with cursor parked low-right (off DayCard).
- **Click 1:** the **streak digit** in Doto. Auto-zoom punches the digit. Hero shot.
- **Click 2:** drag cursor lazily across the FadingRuler to the catalyst tile that ties to the plan. Auto-zoom rides with the cursor — that's the lateral push the script calls "rising arp."
- Stop record. Name `block-E.mp4`.

**5. Block C — Consilium reveal (all cuts · 6s)**
Establishing shot of the whole desk.

- Park cursor center-Sanctum **before** you start the recorder.
- Start Recordly with the window already in tight Sanctum crop (cmd-+ a couple of times in the app, OR start zoomed in via Recordly's manual punch).
- **Don't click anything.** This is the only block where you actively withhold clicks. Instead, **scroll the trackpad outward** (or hit cmd--) to manually pull back to full Consilium. Recordly's auto-zoom respects manual zoom; the pull-out reads as cinematic, not interactive.
- Stop record. Name `block-C.mp4`.

**6. Block J — Pull-back, quiet (60/90 · 6s)**
Same setup as C but reversed in feeling — calm, end-of-piece.

- Already at full Consilium. Start Recordly.
- **No clicks.** Let CAO chat resolve a line in the corner if one's mid-stream; otherwise just hold the wide.
- Stop record. Name `block-J.mp4`.

### Pass 2 — Title cards (one quiet sitting, all black-frame typography)

These don't need Fintheon at all. Make a black-background page (a blank Keynote slide, a fullscreen black HTML page, whatever) and screen-record over it. Recordly's auto-zoom is OFF for these — set Recordly to manual / disable auto-zoom in settings.

- **Block A** — Cold open · 3s · cursor hidden, single Doto digit ticks 00→03 with Solvys-Gold pulse. (Easiest to composite the digit in After Effects/Motion over a 3s black clip; you don't actually record this one in Recordly — just produce a 3s black plate.)
- **Block B** — `PERIPHERAL` → `VISION.` · 4s
- **Block D** — `It's a browser.` / `Bring your own URLs.` · 4s
- **Block G** — `Your sources. / Your seats. / Your desk.` + `Add a URL. It's yours.` · 6s (90s only)
- **Block K** — `Bloomie meets a frosted-obsidian surface.` · 5s
- **Block L** — `Infinite Situational Awareness.` · 5s
- **Block M** — `Speed of the Wire. Without the Terminal Tax.` → `FINTHEON` wordmark + `Coming — Q3 2026.` · 7s

Treat all of Pass 2 as After Effects / Motion / Keynote-export work, not Recordly takes. Recordly is for live UI footage; the title cards live in your motion editor.

## Click-emphasis cheat sheet (which clicks Recordly will hero-zoom)

| Block | Hero click                                                              | Why it matters                                              |
| ----- | ----------------------------------------------------------------------- | ----------------------------------------------------------- |
| C     | (none — manual pull-out)                                                | Establishing shot, no UI emphasis                           |
| E     | Streak digit (Doto)                                                     | Sells "Day plan. Public. Singular." with a numeral          |
| F     | NarrativeFlow card                                                      | The card → drawer slide IS the centerpiece of the 90s       |
| H     | Top wire's IV badge, then cascade-click next wire                       | Velocity flex for "new sheriff"                             |
| I     | Consensus probability digit, then dissent line, then TradingView candle | Three clicks = three zooms = the rhythm of the second build |
| J     | (none — calm hold)                                                      | Pre-final-drop hush                                         |

**Rule of thumb:** if Recordly is going to auto-zoom on every click, **don't click decorative chrome** — only click things that are themselves the message (digits, badges, cards, timestamps). Stray clicks on tab strips, sidebars, or empty space will give you ugly punch-ins on nothing.

## Last housekeeping before you roll

- Hide dock + menubar (`cmd-opt-D`, then enable menubar autohide in System Settings → Control Center).
- Mute notifications (Focus → Do Not Disturb).
- Close every other window. The only thing on the desktop is Fintheon (and TradingView for Block I).
- Cursor color/size: macOS default. Recordly motion-blur handles the rest.
- After every block, **scrub the clip in Finder** — if Recordly stuttered or the auto-zoom landed wrong, retake immediately while the desk state is still set. Cheaper than rebuilding state later.
